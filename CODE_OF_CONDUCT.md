# Code of Conduct

Driftless exists to let people who cannot hand-write code still maintain real
software safely. That only works if the project is a welcoming place — for
non-developers and developers alike. This is a short, standard code of conduct
in the spirit of the [Contributor Covenant](https://www.contributor-covenant.org).

## Our pledge

We pledge to make participation in this project a harassment-free experience for
everyone, regardless of age, body size, disability, ethnicity, gender identity
and expression, level of experience, education, nationality, personal
appearance, race, religion, or sexual identity and orientation.

## Expected behavior

- Be welcoming and patient, especially with non-developers and newcomers.
- Be respectful of differing viewpoints and experiences.
- Give and accept constructive feedback gracefully.
- Focus on what is best for the community and the project.
- Show empathy toward other community members.

## Unacceptable behavior

- Harassment, insults, or derogatory comments, public or private.
- Sexualized language or imagery, and unwelcome sexual attention.
- Publishing others' private information without explicit permission.
- Trolling, deliberate intimidation, or sustained disruption.
- Other conduct that would reasonably be considered inappropriate in a
  professional setting.

## Scope

This code of conduct applies in all project spaces — issues, pull requests,
discussions, and any public space where someone represents the project.

## Enforcement

Report unacceptable behavior to the maintainer
([@mizan0515](https://github.com/mizan0515)) by opening an issue marked clearly
for a maintainer, or — if the report is sensitive — through a
[GitHub Security Advisory](https://github.com/mizan0515/driftless/security/advisories/new)
so it stays private until reviewed.

Reports are reviewed promptly and kept as confidential as the situation allows.
Maintainers may warn, edit or remove contributions, or temporarily or
permanently ban anyone for behavior that violates this code of conduct. We
commit to applying these standards fairly and consistently.
