# Driftless

**드리프트리스는 Claude와 Codex를 위한, 밤새 스스로 좋아지는 자동 유지보수 도구입니다.**

> 한 줄 풀이: *Driftless = 떠내려가지 않음.* 두 도구 프로필이 서로 어긋나지 않고,
> AI가 여러분이 정한 목표에서 벗어나 헤매지 않는다는 뜻이에요.

> **자기 전에 프롬프트 하나 붙여넣기. 일어나면 합쳐진 풀 리퀘스트.**
> 코드는 한 줄도 직접 쓸 필요가 없습니다.
> (*위험하거나 되돌릴 수 없거나 돈이 드는 일은 먼저 물어보라*고 시키는 전체 프롬프트는
> [빠른 시작 4단계](./docs/ko/빠른시작.md)에 있어요.)

## 이게 뭔가요

자기 전에 평범한 말로 된 지시문(프롬프트) 하나만 붙여넣으세요. 자는 동안 AI 정비공이
여러분 저장소의 할 일 목록을 꼼꼼한 동료처럼 처리합니다. 열린 이슈를 읽고, 혼자 해도
안전한 일을 골라서, 브랜치에서 작업하고, 풀 리퀘스트(변경 제안)를 만들고, 모든 안전
검사를 통과한 것만 합쳐요(머지). 아침에 일어나면 합쳐진 풀 리퀘스트가 기다리고 있죠.

보통의 "AI 코딩" 도구는 *여러분이* 운전하길 기대해요. 프롬프트를 쓰고, 바뀐 코드를
읽고, 단계마다 승인해야 하죠. 개발자라면 괜찮습니다. 하지만 드리프트리스는 개발자가
**아닌** 사람을 위한 도구예요. *무엇*을 해야 하는지는 알지만 *어떻게* 컴퓨터에게
시키는지는 모르는 관리자를 위한 거죠.

여러분이 하는 일은 딱 세 가지입니다.

1. **목표를 한 번 말하기.** "프로젝트를 건강하게 유지하고, 쉬운 일들은 끝내고,
   위험한 건 나한테 먼저 물어봐."
2. **잠자기.** AI 정비공이 엄격한 안전 울타리 안에서 혼자 일합니다.
3. **짧은 아침 보고서 읽기.** 여러분의 언어로, 네 가지 분명한 라벨 중 하나로 시작해요.
   완료 / 당신의 결정 필요 / 막힘 / 진행 중.

이 정비공은 스스로 고치고, 브랜치를 만들고, 머지할 수 있어요. 하지만 여러분의 비공개
파일을 건드릴 수 없고, 돈을 쓸 수 없고, 세상에 무언가를 공개할 수 없고, 되돌릴 수 없는
일도 할 수 없습니다. 그런 결정은 항상 짧은 질문으로 여러분에게 돌아와요.

## 여러 도구를 쓰는 건 약점이 아니라 강점이에요

드리프트리스는 일부러 Claude **와** Codex를 둘 다 지원합니다. 신경을 나눠 쓰는 게
아니라 각각의 강점을 쓰는 거예요. **Claude Code**는 만들고 고치는 쪽을 잘하고,
**Codex**(`goal` 모드)는 검토하고 출시하는 쪽에 잘 맞아요. 요즘 에이전트 생태계는
일부러 특정 도구에 매이지 않게 만들어져 있어서(**MCP**와 **AGENTS.md** 같은 공통
표준은 업계가 함께 만든 거예요), 두 도구를 모두 쓰는 건 한 회사에 거는 게 아니라
**생태계의 힘을 빌리는** 거죠. 단일 원본 거울 구조 덕분에 두 번째 도구를 지원하는
비용도 거의 들지 않아요. 공통 부분은 한 번만 작성하니까요.

## 안에 들어 있는 장치 (간단히)

- **하나의 원본, 두 개의 프로필.** 안전 규칙, 용어, 공통 스킬이 `profiles/shared/`
  한 곳에 있고 두 도구가 그걸 읽어요. 한 번만 고치면 둘 다 동시에 좋아지죠. **거울
  일치 검사(mirror-parity gate)** 가 두 프로필이 어긋나지 않았는지 강제합니다.
- **다섯 축 경사 하강.** "더 많은 결과물"이 아니라, 측정 가능한 다섯 축을 시간이
  갈수록 낮춰 가요. 토큰 / 관리자 개입 / 시간 / 돈 / 성능. 매번 측정값을 남겨서
  추세를 추측이 아니라 눈으로 봅니다.
- **강제되는 교훈 승격 사다리.** 배운 교훈은 잊기 쉬운 메모로만 남지 않아요. 무시했을
  때 얼마나 크게 망가지는지에 따라 더 강한 자리로 승격됩니다: 메모 < 필요할 때 부르는
  스킬 < 핵심 규칙 < 훅 < 게이트 스크립트. 끝까지 올라가면 그 교훈을 지키기 전까지
  작업을 기계적으로 실패시키는 검사가 되죠. 시스템은 배울수록 더 깨기 어려워집니다.
- **비공개 파일을 절대 건드리지 않는 격리 가드.** 정적 격리 가드가 모든 변경을 검사해,
  컴퓨터의 전역 `~/.claude`나 `~/.codex`, `.env`, SSH 키, 브라우저 프로필, 비밀스러운
  것은 절대 읽거나 바꾸지 않아요.

## 이 제품은 스스로를 만들었어요

드리프트리스는 시연용 장난감이 아닙니다. 이 도구가 자라난 런타임은 바로 이 밤샘 루프를
**자기 자신의** 저장소에 그대로 썼어요. 이슈가 브랜치가 되고, 브랜치가 풀 리퀘스트가
되고, 풀 리퀘스트가 검토되고 머지됐죠. 자율 정비공이 끌고, 개발자가 아닌 관리자는
울타리가 쳐진 결정만 승인했습니다. 그 원본 저장소의 실제 기록이에요.

- **머지된 풀 리퀘스트 113건**
- **이슈 94건**
- **커밋 114건**
- 그 개발 런타임 기준 두 프로필에 걸친 **Claude 스킬 39개 + Codex 스킬 34개**
  (이 공개 키트는 starter 스킬 12개를 ship — shared 8 + Claude 1 + Codex 1 + 단독 2개
  `overnight-autonomous-work`·`skillopt`. `git ls-files | grep -c SKILL.md`로 직접 세어 보세요)
- 깨끗한 트리에서 통과하는 정적 격리 가드, 스크립트의 ASCII + BOM 없음을 강제하는
  Windows 텍스트 안전 게이트, 프로필 거울 일치 게이트
- 관리자 결정과 에이전트 결정을 이유와 함께 기록한, 날짜가 찍힌 결정 등록부

핵심은 이거예요. 이 루프는 자기 자신을 유지보수할 만큼 진짜라는 것.

## 설치

**먼저 두 가지가 필요해요:** [git](https://git-scm.com/downloads) 과 에이전트 CLI 하나
— [Claude Code](https://docs.anthropic.com/claude-code) 또는
[OpenAI Codex](https://developers.openai.com/codex). 드리프트리스는 에이전트를
*운전*하는 도구이지, 에이전트를 *대신*하는 게 아니에요.

저장소를 받은 뒤 적용합니다 (터미널에 그대로 입력하세요):

```sh
git clone https://github.com/mizan0515/driftless
cd driftless
sh ./install.sh        # macOS / Linux  (Claude / Codex / 둘 다 중 물어봄)
```

```powershell
git clone https://github.com/mizan0515/driftless
cd driftless
.\install.ps1          # Windows PowerShell
```

설치는 작은 모듈 조각들을 여러분의 저장소에 놓아 줍니다. 무거운 프레임워크가 아니라
**여러분이 직접 소유하는 작은 부품들**이에요.
