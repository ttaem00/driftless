# 5분 빠른 시작 (비개발자용)

코딩을 몰라도 됩니다. 이 폴더에서 터미널을 하나 열 수 있고, **Claude Code** 또는
**OpenAI Codex** 중 하나가 설치돼 있으면 됩니다. 둘 다 없으면 하나를 먼저 설치하세요
([Claude Code](https://docs.anthropic.com/claude-code),
[Codex](https://developers.openai.com/codex)). 드리프트리스는 그 도구를 **대신하는 게
아니라 운전**합니다.

> 가장 빠른 길: 이 폴더에서 에이전트가 이미 돌고 있다면, 그냥 **"이 레포 적용해줘"**
> 라고 말하세요. 에이전트가 [apply-driftless](../../profiles/shared/skills/apply-driftless/SKILL.md)
> 절차를 알아서 따릅니다. 아래 수동 단계는 그것을 손으로 하는 것과 같습니다.

## 1단계 — 레포 받기 (1분)

```sh
git clone https://github.com/mizan0515/driftless
cd driftless
```

## 2단계 — 설치기 실행 (1분)

격리된 에이전트 홈을 **이 폴더 안에** 만들고, 컴퓨터의 전역 Claude/Codex 설정은 절대
건드리지 않습니다. 추가로 무언가 설치하기 전에 항상 묻고, 기본 답은 **아니오**입니다.

```sh
sh ./install.sh         # macOS / Linux  (Claude / Codex / 둘 다 중 물어봄)
```
```powershell
.\install.ps1           # Windows PowerShell
```

바꾸기 전에 계획만 보고 싶으면 `sh ./install.sh --dry-run --both` 를 먼저 돌려 보세요.

## 3단계 — 격리 홈으로 에이전트 시작 (1분)

**이 폴더 안에서** 실행하세요. 아래 환경 변수가 CLI를 레포-로컬 격리 홈으로 가리켜서,
컴퓨터의 전역 설정은 건드리지 않습니다 — 그 환경 변수 *자체가* 격리예요. 따로 설치할
런처는 없습니다. (설치기가 여러분이 고른 도구에 맞는 줄을 그대로 출력해 줍니다.)

macOS / Linux:

```bash
CLAUDE_CONFIG_DIR="$(pwd)/.runtime/claude-home" claude    # Claude
CODEX_HOME="$(pwd)/.runtime/codex-home" codex             # Codex
```

Windows (PowerShell):

```powershell
$env:CLAUDE_CONFIG_DIR="$PWD\.runtime\claude-home"; claude   # Claude
$env:CODEX_HOME="$PWD\.runtime\codex-home"; codex            # Codex
```

## 4단계 — 프롬프트 하나 붙여넣기 (30초)

원하는 일을 평이한 말로 이슈 한두 개로 적으세요("문서 오타 고쳐줘", "기여 가이드
추가해줘"). 그다음 이걸 붙여넣으세요:

> 남은 티켓을 갈 수 있는 만큼 밀어줘. 열린 이슈를 살펴서 안전한 병렬 단위로 쪼개고,
> PR을 열고, 모든 게이트를 통과한 것만 머지해줘. 위험하거나 되돌릴 수 없거나 돈이 드는
> 일은 나한테 먼저 물어봐.

**둘 다** 있다면 이 밤샘 루프엔 어느 쪽이든 됩니다. 어림잡아: 일이 **범위가 분명하고
"끝" 신호가 또렷할 때**(예: 열린 PR 전부 리뷰, 릴리스 게이트)는 **Codex**, 목표가 **크거나
열려 있을 때**(예: 백로그 통째 정리)는 **Claude**를 쓰세요. 자세한 구분은
[어떤 도구를 언제 쓰나](../en/codex-and-claude.md), PR 리뷰·릴리스 게이트·분류용
붙여넣기 goal 예시는
[Codex scoped-goal 예시](../../profiles/codex/prompts/infinite-goal.md#scoped-goal-examples-review-and-release-side)
를 보세요.

## 5단계 — 자고 일어나 아침 보고서 읽기 (다음 날 아침)

일어나면 **머지된 풀 리퀘스트**와 짧은 평이한 보고서가 와 있습니다:
**완료 / 당신의 결정 필요 / 막힘 / 진행 중**. 코드가 아니라 결과를 검토합니다.

## 기대하면 안 되는 것 (정직한 한계)

- 돈을 쓰거나, 무언가를 공개하거나, 비공개 파일을 건드리거나, 되돌릴 수 없는 일을
  **묻지 않고** 하지 않습니다.
- 지금 완전히 검증된 건 Windows입니다. macOS/Linux는 설치기는 돌지만 일부 게이트가
  PowerShell 전용이라 [운영체제별 검증 현황](../en/host-evidence-matrix.md)을
  보세요. 검증 안 된 건 약속이 아니라 UNVERIFIED로 표시합니다.
- 이건 **v0.1.0, 첫날**입니다. 작동하고 자기 레포를 공개적으로 유지보수하지만 새 것이니,
  작고 위험 낮은 티켓부터 시작해 신뢰를 키우세요.

## 뭔가 망가지면

안전 게이트를 직접 돌리면 무엇이 잘못됐는지 평이하게 알려줍니다:

```powershell
.\scripts\Test-Containment.ps1 -Path . -AllFiles
.\scripts\Test-ProfileMirrorParity.ps1 -Path .
```

그다음 무엇을 했고 무슨 일이 생겼는지 이슈로 적어 주세요.
