# 스킬 진화 판정표

| 선택 | 기준 |
|---|---|
| update_existing | 기존 트리거와 산출물이 같고 누락된 한 단계만 있다 |
| thin_wrapper | 기존 스킬 여러 개의 순서·게이트만 재사용한다 |
| create_new | 독립 트리거, 독립 산출물, 두 번째 사용 사례가 있다 |
| script_or_gate | 반복적·검증 가능·상태 기반 동작이다 |
| record_only | 재발 가능성은 있으나 아직 일반화 근거가 약하다 |
| no_change | 이미 자산이 막고 있거나 학습 가치보다 유지비가 크다 |

새 자산 전에는 과발화, 미발화, 컨텍스트 증가, 중복 유지보수, 비공개 정보 유출, 제거 가능성을 점검한다.
